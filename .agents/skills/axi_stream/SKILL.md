# 乘法器设计技能总结

## 设计目标
实现支持AXI4-Stream协议的定点数乘法器，64bit输入位宽，支持流水线级数可配置。

## 关键特性
1. **协议兼容性**：完全兼容AXI4-Stream协议标准
2. **数据格式**：支持有符号/无符号定点数乘法，Q32.32格式
3. **流水线架构**：可配置2/4/8级流水线
4. **接口标准**：输入64bit，输出128bit，全握手支持

## 技术规格
- **输入接口**：AXI4-Stream，64bit数据宽度
- **输出接口**：AXI4-Stream，128bit数据宽度  
- **乘法算法**：基4 Booth编码 + Wallace树压缩
- **配置方式**：parameter定义流水线级数、数据格式等

## 架构要点
1. **模块划分**：输入接口、乘法核心、流水线控制、输出接口
2. **状态机设计**：四状态流水线控制（IDLE/PROCESS/STALL/FLUSH）
3. **时序优化**：目标频率500MHz，Wallace树为关键路径
4. **面积估算**：约31k逻辑门，2.6k寄存器，172k um²面积

## 性能指标
- **吞吐量**：1结果/周期（流水线满载）
- **延迟**：2-8周期（可配置）
- **频率**：目标500MHz（典型工艺）
- **背压处理**：全双工握手，支持背压传递

## 可配置性
- 流水线级数：2/4/8级可选
- 数据格式：有符号/无符号，Q格式可配置
- 协议选项：支持TLAST/TKEEP传播等高级功能

## 应用场景
高吞吐量数据处理、数字信号处理、神经网络加速、图像处理等需要连续定点数乘法的场景。
